## Image Guidelines

- **Format:** JPG or PNG
- **Quality:** High resolution (at least 72 DPI)
- **File size:** Keep under 500KB for web performance
- **Alt text:** Always include descriptive captions

## Tools for Creating Images

- **Figma:** For wireframes and prototypes
- **Canva:** For persona cards and journey maps
- **Screenshots:** For before/after comparisons
- **Photo editing:** Photoshop or free alternatives like GIMP
